Glossary of Domain Terms

Domain Terminology Reference

This section helps testers who are not familiar with livestock/cattle farming. Refer to these definitions when encountering unfamiliar terms in the test cases.

Term Definition
Cow / Cattle Farm animal raised for milk production. In this app, each cow has a profile with health details.
Lactation Stage The period when a cow produces milk. Stages: Early (0-100 days after calving), Mid (100-200 days), Late (200+ days), Dry (not producing milk).
Body Condition Score (BCS) A 1-5 scale rating a cow's physical condition. 1 = very thin/emaciated, 3 = moderate/healthy, 5 = obese. Scored in 0.5 increments.
Milk Yield Amount of milk a cow produces per day, measured in liters (L/day).
Milk Fat % Percentage of fat in the milk. Typically 3.0-6.0%. Higher fat = more valuable milk.
Breed The genetic variety of cattle. Examples: Holstein Friesian (high milk yield), Gir (heat tolerant), Sahiwal (indigenous Indian).
Crude Protein (CP%) The percentage of protein in animal feed. Higher CP feeds like cotton seed cake (~24%) cost more.
TDN (Total Digestible Nutrients) A measure of feed energy/nutrition value, expressed as percentage. Higher TDN = more nutritious feed.
Roughage Bulky, high-fiber feed like grass, hay, straw. Low cost but fills the cow's stomach.
Concentrate Nutrient-dense feed like grain, bran, oil cakes. Higher cost but more protein/energy per kg.
Diet Optimization Using algorithms to calculate the ideal mix of available feeds that meets a cow's nutritional needs at minimum cost (or maximum milk production).
Extension Worker A field agent employed by an organization who visits and supports multiple farmers. They can create farmer profiles, record yield data, and generate reports.
Farmer Profile A record in the system representing a cattle farmer. Farmers managed by extension workers have profiles created on their behalf.
Yield Data Daily milk production records for a farmer's cows, typically collected by extension workers.
Organization An entity (NGO, government body, university, cooperative, research institute) that employs extension workers and manages farmer networks.
Onboarding The one-time setup process after registration: selecting language, role, organization, and creating a profile.
PIN A 4-digit Personal Identification Number used instead of a password for authentication.
PWA (Progressive Web App) A web application that can be installed on a device's home screen and works offline, similar to a native app.
Offline Sync Data created while offline is stored locally and automatically uploaded to the server when internet connection is restored.
E.164 Format International phone number format: +[country code][number]. Example: +919876543210 for India.

Test Case Format

Every test case in this document follows the card format shown below. Use this as a reference for reading and executing tests.

TC-EXAMPLE-01 Example Test Case Title Critical

Preconditions

User is logged in and has completed onboarding.

Steps

  1. Navigate to the Home page
  2. Tap the "Add Cow" button
  3. Fill in the cow name field

Expected Result

The cow form page loads with empty fields ready for input.

Acceptance Criteria

  • Form page loads within 2 seconds
  • All required fields are marked
  • Name field accepts text input
